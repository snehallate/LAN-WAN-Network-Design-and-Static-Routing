# LAN-WAN-Network-Design-and-Static-Routing

# Computer Network Projects

This repository contains Computer Network laboratory projects related to LAN/WAN network design and static routing implementation using Cisco Packet Tracer.

# Projects Included

# Project 1: LAN Design and Cost Analysis

Design and implementation of a Local Area Network (LAN) including:

* VLAN configuration
* IP addressing
* Router and switch setup
* Network security
* Cost estimation

# Project 2: Static Routing Implementation

Implementation of:

* LAN and WAN network setup
* Static routing configuration
* Router communication
* IP addressing
* Network connectivity testing
